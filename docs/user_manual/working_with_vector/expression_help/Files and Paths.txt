base_file_name
..............

base_file_name(path)

.. list-table::
   :widths: 15 85
   :stub-columns: 1

* - Description
  - Returns the base name of the file without the directory or file suffix.

  - Arguments
  - **path** - a file path

  - Examples
  - ``base_file_name('/home/qgis/data/country_boundaries.shp')`` → 'country_boundaries'
